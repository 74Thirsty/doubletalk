Frontend Documentation

Purpose

This interface explains how to extend and maintain the customer support frontend surfaces that integrate with the Vantage backend.

Core principles

  • Design for support agents first: clarity over visual novelty.
  • Keep component states deterministic to align with bot workflows.
  • Prefer accessible, keyboard navigable interactions.

Suggested structure

frontend/
├── index.html
├── styles.css
└── app.js

Licensing Guide

The backend repository ships with MIT License terms. When distributing derivative frontend bundles, keep the following in place:

  1. Preserve the original copyright notice.
  2. Include the MIT permission and warranty disclaimer text.
  3. Document third-party package licenses in release artifacts.

Recommended NOTICE text

"This product includes software derived from the Vantage Chatbot project and is distributed under the MIT License."
